# Warriors - GitHub Pages Site

Mobil cihazlar için optimize edilmiş statik web sitesi.

## GitHub Pages'e Yayınlama

1. Bu repository'yi GitHub'a push edin
2. Repository Settings > Pages bölümüne gidin
3. Source olarak "Deploy from a branch" seçin
4. Branch olarak "main" (veya "master") seçin
5. Save butonuna tıklayın
6. Birkaç dakika içinde siteniz `https://[kullanıcı-adı].github.io/[repo-adı]` adresinde yayınlanacak
